# Shift Click Fix
[![](https://img.shields.io/modrinth/v/lvbzCq7u?style=for-the-badge&logo=modrinth)](https://modrinth.com/mod/shiftclickfix)

Do you know that problem when a mod asks you to shift-right-click
a block and it just does not work, because you have something in your offhand?

This mod fixes that problem and prevents use of your offhand when
interacting with a block, so you always use the block before the item.
